## Features
- Twitch chatbot
- Youtube chatbot
- Discord bot
- OBS overlay (SvelteKit)
- User & Coin system
  - Account binding (twitch, youtube, discord) using SvelteKit
  - Getting points
  - Redeem points
- Interactive chat
- Chat TTS on demand (use for repeating question or shoutouts)
